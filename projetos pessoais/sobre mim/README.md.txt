# Projeto: Página "Sobre Mim"

Esta é uma página HTML simples criada como parte do Desafio - Entrega Intermediária do Bootcamp I CEUB 2025.

## Descrição

A página serve como uma breve apresentação pessoal, incluindo:
- Informações sobre minha participação no curso de Ciência da Computação.
- Meus interesses na área de tecnologia.
- Meus objetivos com o curso.

## Tecnologias Utilizadas
- HTML
- CSS (inline, dentro do HTML)

## Como Visualizar
Basta abrir o arquivo `sobre.html` em qualquer navegador web.
